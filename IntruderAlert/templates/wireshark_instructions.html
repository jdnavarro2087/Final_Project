How to use Wireshark to collect pcap files

Wireshark is a powerful network protocol analyzer that allows you to capture and inspect network packets. Follow these steps to collect your own pcap files using Wireshark:

Step 1: Download and Install Wireshark

Visit the Wireshark download page to download the installer for your operatings system follow the instructions provided on the website

Step 2: Open Wireshark

Once installed, open wireshark. You will be presented with a list of network interfaces on your computer.

Step 3: Start Capturing Packets

Select the network interface you want to capture packets from and click the Start Capturing button (the shark fin icon). Wireshark will then begin capturing network traffic

Step 4: Stop Capturing Packets

when you have captured enough packets, click the Stop Capturing Packets button (the red square icon) to stop the capture.

Step 5: Save the Capture File

To Save your captured packets, go to File > Save As. Choose a location and a file name, then click Save. Your capture file will be saved with a .pcap extension.

For more detailed instructions and advanced features, refer to the Wireshark User's Guide.
